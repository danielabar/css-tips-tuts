These form elements have custom styling

First, hide the original form elements


    input[type="checkbox"],
    input[type="radio"] {
      display: none;
    }

Then target the label to build the custom element


    input[type="checkbox"] + label:before {
      background-color: #D668CD;
      border: 3px solid #756D91;
      content: "";
      display: inline-block;
      font-size: 16px;
      height: 20px;
      width: 20px;
      line-height: 20px;
      margin: -2px 6px 0 0 ;
      vertical-align: middle;
    }

Finally, target the label when the checkbox is checked, to display a check symbol


    input[type="checkbox"]:checked + label:before {
      content: "\2713";
      text-align: center;
    }

Same technique is used for the radio buttons


    input[type="radio"] + label:before {
      background-color: #D668CD;
      border: 3px solid #756D91;
      border-radius: 12px;
      content: "";
      display: inline-block;
      font-size: 18px;
      height: 20px;
      width: 20px;
      line-height: 20px;
      margin: -2px 6px 0 0 ;
      vertical-align: middle;
    }

    input[type="radio"]:checked + label:before {
      content: "\2022";
      text-align: center;
    }
