Hi, this is a layered paper effect.

    .paper {
      background-color: #fafafa;
      border: 1px solid #f2f2f3;
      border-bottom: none;
      padding: 1em;
      text-align: center;

      box-shadow:
        0 1px 1px rgba(0, 0, 0, .15),
        0 10px 0 -5px #eee,
        0 10px 1px -4px rgba(0, 0, 0, .15),
        0 20px 0 -10px #eee,
        0 20px 1px -9px rgba(0, 0, 0, .15);
    }
