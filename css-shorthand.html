This paragraph is styled with the long form of font styling


    p {
      font-style: italic;
      font-weight: bold;
      font-size: 13px;
      line-height: 1.5em;
      font-family: Arial, sans-serif;
    }

This paragraph is styled with the shorthand font styling


    p {
      font: italic bold 13px/1.5em Arial, sans-serif;
    }

This paragraph is styled with the long form of padding


    p {
      padding-top: 1em;
      padding-bottom: 1em;
      padding-left: 1em;
      padding-right: 1em;
    }

This paragraph has 1em of padding all around


    p {
      padding: 1em;
    }

    p {
      padding: 1em 2em;
