The key is using position: fixed


    .back-to-top {
      color: #DAF6F6;
      background-color: #332E2E;
      background-color: rgba(51, 46, 46, 0.85);
      position: fixed;
      bottom: 2em;
      right: 2em;
      line-height: 3em;
      border: 1px solid #CCC;
      border-radius: 3px;
      padding-left: 20px;
      padding-right: 20px;
      text-transform: uppercase;
      font-size: 1em;
      text-decoration: none;
    }

    .back-to-top:hover {
      color: #D319BD;
    }
