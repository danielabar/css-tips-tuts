Start with width and height of 0


    .arrow {
      height: 0;
      width: 0;
      margin-bottom: 1em;
    }

Up arrow is made with opaque bottom border, and transparent left and right borders


    .up {
      border-left: 10px solid transparent;
      border-right: 10px solid transparent;
      border-bottom: 52px solid red;
    }

Down arrow is made with opaque top border, and transparent left and right borders


    .down {
      border-left: 30px solid transparent;
      border-right: 30px solid transparent;
      border-top: 52px solid blue;
    }

Left arrow is made with opaque right border, and transparent top and bottom borders


    .left {
      border-top: 30px solid transparent;
      border-bottom: 30px solid transparent;
      border-right: 52px solid green;
    }

Right arrow is made with opaque left border, and transparent top and bottom borders


    .right {
      border-top: 30px solid transparent;
      border-bottom: 30px solid transparent;
      border-left: 52px solid yellow;
    }

Start with right arrow and rotate 90 °


    .right-rotated {
      border-top: 30px solid transparent;
      border-bottom: 30px solid transparent;
      border-left: 53px solid purple;
      -webkit-transform: rotate(90deg);
      transform: rotate(90deg);
    }
