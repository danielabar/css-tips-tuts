This is a regular form with default browser styling

This form has the inputs custom styled


    input[type="text"],
    textarea {
      border: 2px solid #771A1A;
      border-radius: 5px;
      outline: none;
      padding: 0.5em;
    }
This form has the labels moved on top of the inputs


    label {
      display: block;
      margin-bottom: 6px;
    }
This form has custom styled hover and focus states


    input[type="text"]:hover,
    textarea:hover {
      box-shadow: 2px 2px 5px #6650DA;
      transition: box-shadow .2s ease-in-out;
    }
    input[type="text"],
    textarea {
      transition: box-shadow .2s ease-in-out;
    }
    input[type="text"]:focus,
    textarea:focus {
      box-shadow: 2px 2px 5px #6650DA;
      border-color: #74b9ef;
    }
